## 4. Продуктовая логика

### Пресеты фильтра периода
```
Оптимальный набор для фильтра:

- Сегодня — спонтанные поездки
- Завтра — краткосрочное планирование
- В эти выходные — суббота/воскресенье
- Ближайшие 7 дней — дефолт, если день не выбран
- Ближайший месяц — корпоративы, ДР, тренировки заранее
- Выбрать даты — календарь для произвольного диапазона
```

### Уточнение логики пресетов
```
Сегодня / Завтра

На этой неделе: слоты с текущего момента до конца календарной недели

В эти выходные: ближайшие сб+вс.
Если открыли в сб/вс — текущие выходные; в пн — следующие

Ближайший месяц: rolling window +30 дней от сегодня

Выбрать даты
```

### Фикстуры для теста фильтров
```
Добавь карточки с разными маршалами-инструкторами и датами (всего 5 маршалов),
чтобы можно было проверить каждый пресет периода.
```
