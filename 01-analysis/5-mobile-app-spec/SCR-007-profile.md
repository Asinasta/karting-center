# SCR-007 — Профиль

## API

- `getProfile`
- `updateProfile`
- `sendPhoneChangeOtp`
- `verifyPhoneChange`
- `deleteAccount`
- `registerPushToken`

## UI

Имя, телефон, управление push-разрешением, выход, удаление аккаунта, ссылки на документы.

Имя редактируется через `updateProfile`. Смена телефона открывает отдельный OTP-flow: отправка кода на новый номер, подтверждение, затем обновление профиля. Старый номер остаётся активным до успешного подтверждения.

## Flutter-заметки

При выходе токены очищаются из secure storage. При удалении аккаунта токены очищаются только после успешного ответа API; backend применяет обычную политику отмен к активным броням.
