# LOGIC-008 — Push / in-app напоминание «Оцените маршала»

## Где используется

SCR-005, LOGIC-007 (расширение).

## API

- `GET /notifications` — pending-уведомления для текущего клиента

## Правило `rate_marshal`

Показывать напоминание, если:

- бронь принадлежит клиенту
- `can_rate_marshal` (см. LOGIC-006)
- `now >= slot.start_at + duration_min` (длительность заезда из track_config, иначе 15 мин)
- оценки ещё нет

## Клиент

- после загрузки «Мои записи» запрашивает `/notifications`
- для `rate_marshal` показывает баннер / диалог с переходом на SCR-006
- при подключении FCM тот же payload уходит как push
