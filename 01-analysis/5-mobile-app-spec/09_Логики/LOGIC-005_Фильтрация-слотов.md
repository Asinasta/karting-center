# LOGIC-005 — Фильтрация слотов

## Где используется

SCR-002, BS-001.

## Параметры API

- `date_from`, `date_to`
- `track_config_type[]`
- `marshal_id[]`
- `only_available`

## Правила

Внутри одной группы значения объединяются по OR, между группами — по AND. По умолчанию фильтры не заданы, период — ближайшие 7 дней.
