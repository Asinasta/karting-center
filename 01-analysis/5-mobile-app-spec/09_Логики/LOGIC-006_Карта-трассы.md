# LOGIC-006 — Карта трассы

## Где используется

SCR-003, SCR-006, BS-004.

## Правила

- API отдаёт `meeting_point`, координаты и `geometry`.
- Flutter-клиент показывает pin места сбора и схему трассы.
- При недоступности карты показывается текстовый fallback.

## Flutter-заметки

Конкретный плагин карт выбирается при реализации. API-ключ хранится в конфигурации окружения, не в репозитории.
