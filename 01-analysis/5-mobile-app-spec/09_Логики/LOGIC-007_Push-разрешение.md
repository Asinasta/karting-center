# LOGIC-007 — Push-разрешение

## Где используется

BS-002, SCR-007.

## Правила

- Запрос системного разрешения показывается после первой успешной брони.
- При разрешении Flutter получает push token и вызывает `registerPushToken`.
- Если пользователь отказал, приложение работает без push.
- Тайминги напоминаний `[24, 2]` — предположение и должны приходить из API/конфига.

## Flutter-заметки

Использовать FCM/APNs через выбранный Flutter-плагин. Не блокировать основной сценарий записи, если push недоступен.
