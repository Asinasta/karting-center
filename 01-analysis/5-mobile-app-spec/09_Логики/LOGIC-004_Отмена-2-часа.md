# LOGIC-004 — Отмена и правило 2 часов

## Где используется

SCR-006, BS-003.

## Правила

- Порог ранней отмены 2 часа — предположение, не прямой факт из брифа.
- Ровно 2 часа до старта считается ранней отменой.
- Тип отмены определяет сервер.
- Ранняя отмена возвращает места и прокатную экипировку.
- Поздняя отмена ставит `late_cancel` и не освобождает места.

## Flutter-заметки

Клиент может показывать предупреждение на основе локального времени, но должен принять статус из ответа `cancelBooking`.
