# LOGIC-008 — Состояния экрана

## Где используется

Все экраны.

## Состояния

- Loading.
- Content.
- Empty.
- Error with retry.
- Offline stale.

## Правила

Мутации (создание и отмена брони) офлайн запрещены. Кэш можно показывать только для read-only данных: список слотов, карточки и брони, с пометкой «данные могли устареть».
