# SCR-002 — Список заездов

## API

- `listSlots`: `GET /slots` без авторизации

## UI

Публичный список карточек слотов, кнопка фильтров, таб-навигация. По умолчанию период 7 дней.

Неавторизованный клиент может смотреть список и карточки. При переходе к записи или персональным разделам приложение запускает OTP-вход.

## Состояния

Loading, Content, Empty «Пока нет доступных заездов», Error/Retry, Offline stale.

## Применяемые логики

LOGIC-002, LOGIC-005, LOGIC-008.
