# SCR-004 — Оформление записи

## API

- `createBooking`: `POST /bookings`

## UI

Сводка заезда, выбор 1–3 мест (лимит 3 — предположение/R-013), выбор экипировки по каждому месту, итоговая цена, кнопка подтверждения.

## Технические правила

Flutter-клиент генерирует `Idempotency-Key` на попытку создания брони и повторяет запрос с тем же ключом при неопределённом результате.

## Применяемые логики
