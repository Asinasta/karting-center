# SCR-004 — Оформление записи

## API

- `createBooking`: `POST /bookings`

## UI

Сводка заезда, выбор мест степпером (до всех свободных), компактный выбор экипировки счётчиками «своя / прокат», локальный preview цены, кнопка подтверждения.

После успешного `createBooking` экран успеха показывает только финальный `price_total` из API. Локальный preview не является источником истины.

## Технические правила

Flutter-клиент генерирует `Idempotency-Key` на попытку создания брони и повторяет запрос с тем же ключом при неопределённом результате.

Если форма открыта из публичной карточки без активной сессии, сначала выполняется OTP-вход, затем создание брони.
