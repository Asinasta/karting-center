# Хорошие промпты для проверки требований

- Проверь требования на неподтверждённые числа и перечисли, где нужна пометка «предположение».
- Найди расхождения терминов между FR, use cases, design brief, API и mobile spec.
- Проверь, не попали ли в клиентский MVP функции маршала, владельца или админки.
- Проверь, покрыты ли все Must FR пользовательскими историями и use cases.
