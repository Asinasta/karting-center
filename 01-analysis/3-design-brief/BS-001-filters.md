# BS-001 — Фильтры

## Цель

Помочь клиенту быстро сузить список заездов.

## Поля фильтра

- Период дат: сегодня, завтра, на неделю вперёд, в эти выходные, ближайший месяц, произвольный диапазон.
- Тип трассы: новичковая, опытная.
- Маршал: множественный выбор, если справочник не пуст.
- Переключатель «Только свободные».

## Поведение

Внутри одной группы значения объединяются по OR, между группами — по AND. Кнопки: «Применить», «Сбросить».

## Источники

FR-38, UC-3.
