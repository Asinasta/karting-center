# SCR-007 — Профиль клиента

## Цель

Дать клиенту управлять личными данными и сессией.

## Состав

- Имя.
- Номер телефона.
- Настройки push-разрешения с переходом в системные настройки.
- Ссылки: политика конфиденциальности, условия.
- Выход из аккаунта.
- Удаление аккаунта.

## Особые состояния

- Смена телефона требует OTP.
- Удаление аккаунта требует подтверждения и объясняет последствия для активных броней.

## Источники

FR-47, FR-48, FR-49.
