# SCR-002 — Список заездов

## Цель

Показать клиенту ближайшие заезды и дать быстро выбрать подходящий слот.

## Состав карточки слота

- Дата и время старта.
- Название и тип конфигурации трассы.
- Маршал.
- Цена от `price`.
- Свободные места / всего мест.
- Доступность прокатной экипировки.
- Статус: доступен, мест нет, отменён центром.

## Действия

- Открыть фильтры.
- Открыть карточку заезда.
- Перейти в «Мои записи» / «Профиль» через навигацию.

## Empty / Error

- Нет расписания: «Пока нет доступных заездов».
- Фильтр без результатов: «Ничего не найдено, измените фильтры».
- Offline stale: показывается кэш с предупреждением.

## Источники

FR-9, FR-38, US-2, UC-3.
