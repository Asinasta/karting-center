# SCR-004 — Оформление записи

## Цель

Позволить клиенту оформить бронь на выбранный заезд.

## Состав

- Сводка заезда.
- Счётчик мест: 1–3; лимит 3 — предположение/R-013.
- Для каждого места: своя / прокатная экипировка.
- Итоговая цена `price_total` из API.
- Пояснение: оплата на месте наличными или переводом.

## Ошибки

- Мест уже нет.
- Не хватает прокатной экипировки.
- Заезд отменён центром.
- Нет сети.

## Источники

FR-10–FR-15, FR-45, UC-1.
