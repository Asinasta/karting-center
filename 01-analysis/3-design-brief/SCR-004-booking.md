# SCR-004 — Оформление записи

## Цель

Позволить клиенту оформить бронь на выбранный заезд.

## Состав

- Сводка заезда.
- Счётчик мест: 1…`min(free_seats, capacity_cap)`; кнопка «Все свободные».
- Экипировка: компактные счётчики «своя / прокат» и пресеты.
- Для каждого места: своя / прокатная экипировка.
- Локальный preview цены до подтверждения; финальный `price_total` показывается после создания брони.
- Пояснение: оплата на месте наличными или переводом.

## Ошибки

- Мест уже нет.
- Не хватает прокатной экипировки.
- Заезд отменён центром.
- Уже есть активная бронь на этот заезд.
- Заезд уже начался.
- Нет сети.

## Источники

FR-10–FR-15, FR-45, UC-1.
