# SCR-001 — Регистрация / вход

## Цель

Дать клиенту быстрый вход без пароля: телефон → SMS-код → имя для нового пользователя. Если вход запущен из записи на заезд, после успеха вернуть клиента к выбранному заезду.

## Состав

- Поле телефона.
- Кнопка «Получить код».
- Экран ввода OTP.
- Поле имени для нового пользователя.
- Ссылки на политику конфиденциальности и согласие.

## Состояния

- Ошибка формата телефона.
- Неверный код.
- Лимит попыток / повторная отправка позже.
- Код истёк / SMS не пришла.
- Сеть недоступна.

## Источники

FR-1, FR-2, FR-43, NFR-18, NFR-19, NFR-20.
