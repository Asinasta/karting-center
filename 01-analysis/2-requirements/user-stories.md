# User stories

> Этап 2. Пользовательские истории в формате «Как <роль>, я хочу <действие>, чтобы <ценность>».

| ID | Как | Хочу | Чтобы | Критерии приёмки | FR |
| :-- | :-- | :-- | :-- | :-- | :-- |
| US-1 | Клиент | зарегистрироваться по имени и телефону | быстро начать пользоваться приложением | После ввода телефона, SMS-кода и имени клиент попадает в приложение; пароль не нужен | FR-1, FR-2, FR-43 |
| US-2 | Клиент | видеть список ближайших заездов | выбрать подходящий слот | По умолчанию показаны слоты на 7 дней, сортировка по старту, есть empty state | FR-9 |
| US-3 | Клиент | фильтровать заезды | быстро найти подходящий вариант | Фильтры по дате, типу трассы, маршалу и наличию мест применяются и отображаются | FR-38 |
| US-4 | Клиент | открыть карточку заезда | понять детали перед записью | Видны конфигурация, маршал, цена, места, прокат экипировки, место сбора | FR-9a, FR-44 |
| US-5 | Клиент | записаться на заезд | забронировать место | После подтверждения создана бронь, она появилась в «Моих бронированиях» | FR-10 |
| US-6 | Клиент | выбрать свою или прокатную экипировку | приехать со своим комплектом или взять напрокат | Для каждого места выбран тип экипировки; прокат уменьшает `free_rental_gear` | FR-11, FR-14 |
| US-7 | Клиент | записать компанию одной бронью | не оформлять каждого отдельно | Можно выбрать 1–3 места; данные гостей не собираются | FR-12 |
| US-8 | Клиент | быть защищённым от овербукинга | не приехать на занятый слот | При нехватке мест/экипировки бронь отклоняется с понятной ошибкой | FR-13, FR-15 |
| US-9 | Клиент | видеть свои бронирования | контролировать планы | Видны предстоящие, прошедшие и отменённые брони со статусами | FR-35a |
| US-10 | Клиент | отменить бронь | освободить место, если планы изменились | Отмена только целиком; ранняя освобождает места, поздняя фиксируется без освобождения | FR-16, FR-17, FR-18 |
| US-11 | Клиент | видеть цену | понимать стоимость до приезда | Цена показана в карточке, оформлении и подтверждении; оплата офлайн | FR-30, FR-45 |
| US-12 | Клиент | получать напоминания | не забыть о заезде | При разрешённом push приходят напоминания; без разрешения приложение работает штатно | FR-33 |
| US-13 | Клиент | видеть отмену центром | не приехать на отменённый заезд | Бронь получает статус «Отменён центром», указана причина, приходит push | FR-46 |
| US-14 | Клиент | управлять профилем | держать актуальные данные | Можно изменить имя/телефон, выйти, удалить аккаунт | FR-47, FR-48, FR-49 |
| US-15 | Клиент | видеть понятные ошибки | понимать, что делать дальше | Ошибки нехватки мест, сети, OTP, отменённого слота локализованы и предлагают действие | FR-10, FR-16 |
