# Чек-лист цифровой гигиены

Перед передачей в разработку проверить:

- [ ] Все числа имеют источник: бриф, R-решение или пометку «предположение».
- [ ] В MVP нет интерфейса маршала и владельца.
- [ ] Flutter зафиксирован в NFR и mobile-app-spec.
- [ ] OpenAPI использует `marshals`, а не `instructors`.
- [ ] Термины единообразны: заезд, слот, конфигурация трассы, маршал, экипировка.
- [ ] Use cases содержат альтернативные и ошибочные потоки.
- [ ] `createBooking` использует `Idempotency-Key`.
- [ ] Карта трассы имеет fallback без интерактивной карты.
- [ ] Онлайн-оплата, оценки маршалов, авто-погода и лояльность вынесены в Phase 2.
